# McCity

A large-scale downloadable Minecraft city project focused on exploration, atmosphere, and detail.

## About

McCity is a high-detail urban Minecraft environment built for discovery and exploration. The map features:

* 50+ furnished buildings
* Underground tunnel systems
* Handmade terrain
* Detailed interiors
* Quest content
* Java & Bedrock support

The project is designed as a public downloadable city while preserving creator attribution and community sharing rules.

## Features

* Massive explorable city
* Functional transit areas
* Unique districts
* Hidden locations and secrets
* Cinematic environments
* Optimized for survival and exploration gameplay
